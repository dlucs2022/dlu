# DLU AI
##平台前端

## 部署
### 1. 安装docker和docker compose
> Docker 官方文档：https://docs.docker.com/get-docker/
### 2. 下载项目源码
```shell
git clone https://github.com/dlucs2022/dlu.git
```
### 3. 切换目录，构建启动项目
```shell
docker compose up -d
```
#### 4. 测试
> 访问 http://部署主机IP:80
### 5. 停止项目
```shell
docker compose stop
```
### 6. 启动项目
```shell
docker compose start
```
### 7. 重启项目
```shell
docker compose restart
```
### 8. 清除项目
```shell
docker compose down
```
